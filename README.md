# Bitat Projesi

## Proje Açıklaması

Bu proje, C# Windows Forms kullanılarak geliştirilmiş basit bir masaüstü uygulamasıdır.

Uygulama, temel kullanıcı arayüzü işlemleri ve SQL Server veritabanı bağlantısı üzerine çalışmaktadır.

---

## Özellikler

* Windows Forms tabanlı kullanıcı arayüzü
* SQL Server veritabanı bağlantısı
* Veri ekleme işlemleri
* Temel form işlemleri (ekleme / giriş ekranları)

---

## Kullanılan Teknolojiler

* C#
* Windows Forms
* SQL Server

---

## Proje Amacı

Bu proje, Windows Forms ve SQL Server bağlantısı konularında temel seviyede pratik yapmak ve veri tabanı mantığını öğrenmek amacıyla geliştirilmiştir.
